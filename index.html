Projects I'm proud of

My Bachelor Project

Le Projet

This project's aim is to combine in one web app all offers of online sellers in Moldova. The web app allows filtering and sorting of products from different marketplaces. Furthermore, the user can add the products to favorites and monitor the change of their prices.

It uses dockerized web scrapers of 4 websites. They and the backend had been created using nest.js framework with axios and cheerio libraries. The frontend had been written in vue.js framework with vuetify, vuex, vue-router, vuex-persistedstate libraries. Last but not least is Integration of Muscula for errors logging and monitoring.

Project's repository

Technologies used include:

  • Docker,
  • Nest.js,
  • Vue.js,
  • Go,
  • Muscula
